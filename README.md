# Focus Assistant

- A personal productivity assistant built with React.js that helps you plan your day, stay focused using the Pomodoro technique, and track your weekly performance with smart suggestions.

# Features

- Task management: add, complete, and delete daily tasks
- Pomodoro timer for focus sessions
- Weekly summary with visual charts
- Smart suggestions based on completed tasks
- Local storage support to keep data after refresh

# Tech Stack

- React.js
- CSS3
- Recharts (for data visualization)

# Concept

- This app is based on time management principles combining task planning, Pomodoro focus cycles, and weekly self-assessment.  
  It helps users maintain consistency and track personal growth over time.

# Setup Instructions

1. Clone this repository
   git clone https://github.com/gamzefidan/productivity-assistant.git

2. Navigate to the project folder  
   cd productivity-assistant

3. Install dependencies
   npm install

4. Run the app
   npm start
